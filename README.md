# Skills

Hand-written Agent Skills (forked from antfu/skills).

## Installation

```bash
pnpx skills add huguangju/skills --skill <skill-name>
```

## Available Skills

| Skill | Description | Install |
|---|---|---|
| **deepwiki** | Generate DeepWiki-style documentation from source code and integrate with VitePress. | `pnpx skills add huguangju/skills --skill deepwiki` |
| **docs-simplify** | Simplify technical/product docs without changing meaning. | `pnpx skills add huguangju/skills --skill docs-simplify` |
| **vitepress-migration** | Migrate documentation into VitePress while preserving structure, links, and assets. | `pnpx skills add huguangju/skills --skill vitepress-migration` |

## Usage

```text
Use the <skill-name> skill to ...
```
